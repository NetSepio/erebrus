About this node

This is an Erebrus DePIN VPN node. It serves a WireGuard fast path plus DPI-resistant stealth carriers (VLESS+REALITY over TLS, Hysteria2 over QUIC) so the same tunnel keeps working on networks that throttle or block WireGuard. The node keeps no traffic logs; this page shows only coarse aggregates. Region: … · Version: ….
